# Procedures | Books | Kognitos Documentation

The following documentation is forSAP v2.4.1(BDK).

Ensure that you have installed or connected theSAPbook and created a new playground before using these automation procedures.

## hashtagto retrieve some entities fromsap

Retrieves the list of available SAP entities for a given API service.

Input Concepts

api service

The API service to retrieve the entities from.

(no default)

Output Concepts

the list of available SAP entities in the Service

## hashtagto retrieve some services fromsap

Retrieves the list of available SAP services.

Output Concepts

the list of available SAP services

## hashtagto upload an attachment tosap

Upload an attachment to SAP and returns the created entity data as a dictionary.

Input Concepts

The attachment to upload.

(no default)

The name of the file.

(no default)

business object type

The business object type.

(no default)

linked sap object key

The linked SAP object key.

(no default)

Output Concepts

The created entity data.
