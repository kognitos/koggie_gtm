# Getting Dates and Times | Dates & Times | Kognitos Documentation

### hashtag1. Get the Current Date

Returns the current date inMM-DD-YYYYformat, such as01-23-2025.

### hashtag2. Get the Current Time

Returns the current time inHH:MM:SS.ssssssformat, such as10:29:58.348854.*

> *By default, the current time is returned in Coordinated Universal Time (UTC). To get the current time in a specific time zone, seeSetting the Time Zone.

*By default, the current time is returned in Coordinated Universal Time (UTC). To get the current time in a specific time zone, seeSetting the Time Zone.

### hashtag3. Get the Current Year

Returns the current year in the formatYYYY, such as2025.

### hashtag4. Get the Current Month

Returns the current month by name, such asFebruary.

### hashtag5. Get the Current Day

Returns the current day by name, such asThursday.

### hashtag6. Get the Current Hour

Returns the current hour in 12-hour format along with AM/PM. For example:02PM.
