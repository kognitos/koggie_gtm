# Convert Excel to PDF | Excel (v2) | Kognitos Documentation

### hashtagOverview

This procedure converts an Excel file to PDF format. If the Excel file contains multiple worksheets, all worksheets are combined into a single PDF document. The conversion handles .xlsx, .xls, and .xlsm file formats.
