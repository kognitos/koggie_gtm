# Google Drive | Books | Kognitos Documentation

The following documentation is forGoogle Drive v2.0.1(BDK).

## hashtagOverview

Google Drive offers cloud storage and file synchronization with seamless integration across Google Workspace. This integration enables automated file management, sharing workflows, and document organization processes. Streamline file collaboration and maintain synchronized access to important documents.

## hashtagConnectivity

This section outlines the available methods for connecting to the Book, along with the required configuration details for each.

### hashtagConnect using Client Email, Token URI and Private Key

Gets the credentials from the service account keys.

Client Email

The client email of the service account registered in the Google Cloud Console.

The token URI of the Google Cloud Console.

Private Key

The private key of the service account registered in the Google Cloud Console.

### hashtagContinue with Google

To connect to theGoogle Drivebook, click onContinue with Google. This will redirect you to Google's secure authentication page where you can sign in with your Google account and authorize the application. Once connected, the Kognitos agent will have access to your Google Drive while keeping your credentials secure.
