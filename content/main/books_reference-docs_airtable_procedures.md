# Procedures | Books | Kognitos Documentation

The following documentation is forAirtable v2.0.0(BDK).

Ensure that you have installed or connected theAirtablebook and created a new playground before using these automation procedures.

## hashtagto create a record in a (table)

Creates a new record in a table

Input Concepts

The Airtable table to get records from

airtable table

(no default)

record data

A dictionary with the fields and values of the new record

(no default)

Output Concepts

the newly created record

airtable record

Create a record in a table

## hashtagto get some (base's tables)

Gets a list of all tables from a specific base

Input Concepts

The Airtable base to get tables from

airtable base

(no default)

Output Concepts

base's tables

A list of tables

airtable table

Get all tables from a base

Get a single table from a base

## hashtagto get some (bases)

Gets a list of all bases

This procedure can get a list of all bases, as well as getting a single base filtering by its name or id.

Output Concepts

A list of bases

airtable base

Get all bases

Get a single base by name
