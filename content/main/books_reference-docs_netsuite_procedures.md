# Procedures | Books | Kognitos Documentation

The following documentation is forNetSuite v2.1.3(BDK).

Ensure that you have installed or connected theNetSuitebook and created a new playground before using these automation procedures.

## hashtagto retrieve a file's information fromnetsuite

Get a file's information from NetSuite's File Cabinet, using the SOAP API.

Input Concepts

The ID of the file to retrieve information for

(no default)

Output Concepts

The file's information

(no default)
