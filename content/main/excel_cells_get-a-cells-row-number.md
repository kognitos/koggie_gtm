# Get a Cell's Row Number | Excel (v2) | Kognitos Documentation

### hashtagOverview

This procedure retrieves the row number of a specific cell in an Excel worksheet, returning a numeric value such as 1, 2, 3, etc. It’s useful in automation workflows that rely on dynamic row positions. For example, when identifying the location of data, iterating through rows, or applying logic based on where a cell appears in the sheet.
