# Get a Column | Tables | Kognitos Documentation

### hashtagOverview

This procedure gets a column from a table, given the column's name. The data is returned as a single-column table. To get a list of values from a column instead, see the procedureGet a Column's Values.

### hashtagInput Concepts

column name

The name of the column to retrieve.

### hashtagOutput Concepts

The specified column from the table.
