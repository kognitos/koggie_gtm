# Procedures | Books | Kognitos Documentation

The following documentation is forConfluence v1.0.0(BDK).

Ensure that you have installed or connected theConfluencebook and created a new playground before using these automation procedures.

## hashtagto create a page in a (space)

Creates a page in Confluence

Input Concepts

The Confluence space where the page will be created

confluence space

(no default)

The title of the page to create

(no default)

The body content of the page in wiki format

(no default)

Output Concepts

The created ConfluencePage instance

confluence page

Retrieve all pages

## hashtagto retrieve some (pages) fromconfluence

Retrieves pages from Confluence

Output Concepts

the list of pages

confluence page

Retrieve all pages

Retrieve a space by its title.

the list of spaces

confluence space
