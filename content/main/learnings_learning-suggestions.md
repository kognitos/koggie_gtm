# Learning Suggestions | Kognitos Documentation

Within theGuidance Center, you may seeSuggestionslisted below an exception. These suggestions represent learning techniques provided by the system as potential solutions, ranked by how closely the current document matches the document where the learning technique was previously applied.

### hashtagHow to Apply Learning Suggestions

- Hover over aTechniqueto see the confidence level displayed as a percentage. This score measures how similar two documents are.
