# Get a Column Number | Tables | Kognitos Documentation

### hashtagOverview

Retrieves the column number (index) of the specified column in a table.

Note:Columns are numbered starting with the number 0.

### hashtagInput Concepts

The column whose index is to be retrieved.

### hashtagOutput Concepts

column number

The number of the column (index).

### hashtagExamples

#### hashtag1. Example 1

In this example,the columnrefers to theAddresscolumn in the table below.

+ 1 (415) 691 9426

123 east street

San Francisco

+ 1 (415) 691 1234

321 west street

+ 1 (415) 691 4595

213 north street

San Francisco

Note: This sample table can be generated using thecreate a sample tableprocedure.

Since the columns are numbered starting at 0, theAddresscolumn has a column number (index) of 2.
