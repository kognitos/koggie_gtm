# Procedures | Books | Kognitos Documentation

The following documentation is forAzure Service Bus v1.3.2(BDK).

Ensure that you have installed or connected theAzure Service Busbook and created a new playground before using these automation procedures.

## hashtagto receive a (message) from a queue

Receive the latest message from an Azure Service Bus queue.

Input Concepts

The name of the Azure Service Bus queue

(no default)

max wait time

Maximum time to wait for a message in seconds

(no default)

Output Concepts

The latest message received from the queue

Receive a message from Azure Service Bus queue

Receive a message from Azure Service Bus queue with a maximum wait time of 60 seconds

The message to be sent

(no default)

The name of the Azure Service Bus queue

(no default)
