# Set a Cell's Value | Excel (v2) | Kognitos Documentation

### hashtagOverview

This procedure sets the value of a cell in a worksheet to a specified text or formula. It's useful for dynamically updating spreadsheets with new data or calculations.

Onlytextvalues can be used to set the value of a cell for this procedure.
