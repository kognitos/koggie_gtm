# Create a Sample Table | Tables | Kognitos Documentation

## hashtagOverview

This procedure generates a sample table. The generated table will have the columnsName,Number,Address, andCityand three rows of predefined data:

+ 1 (415) 691 9426

123 east street

San Francisco

+ 1 (415) 691 1234

321 west street

+ 1 (415) 691 4595

213 north street

San Francisco

This procedure will consistently generate the same table. It can be easily expanded, modified, or used as a foundation for further data processing.

## hashtagOutput Concepts

A table containing sample data with columns forName,Number,Address, andCity.
