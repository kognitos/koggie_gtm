# @config | Books | Kognitos Documentation

## hashtagOverview

Theconfigdecorator is used to decorate a property in a Book class that defines default configuration values.

## hashtagSyntax

## hashtagKeyword Arguments

Specifies the name of the configuration. If not provided, the function name or value from*argsis used.

default_value

Specifies the default value for the configuration. If not provided, the default isNone.
