# Get a Cell's Formula | Excel (v2) | Kognitos Documentation

### hashtagOverview

This procedure finds and retrieves the formula from a specific cell in an Excel worksheet. If the cell does not contain a formula, it returns "No formula." This is useful in automation workflows that need to access and interpret formulas.
